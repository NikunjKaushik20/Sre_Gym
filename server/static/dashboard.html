Core Capabilities
⚡

Instant Reset

Reset the entire cluster state in <100ms for rapid training iterations and epoch efficiency. Seed-reproducible fault injection ensures fair evaluation.

📊

Action Space

Six structured action types: query_logs, query_metrics, submit_diagnosis, apply_remediation, escalate, and submit_postmortem.

🎯

Reward Shaping

Strict deterministic graders — no LLM judges. Exact canonical cause names required. Terminal-only scoring prevents reward hacking.

🕸️

Dynamic Graph

Agents perceive infrastructure as a directed dependency graph. Partial observability expands as services are queried — no free information.

🎣

Red Herring Injection

Decoy alerts and metrics with −0.40 diagnosis penalty. Agents that chase noise are explicitly punished, separating strong from weak models.

⚔️

Adversarial Grading

Substring guessing yields zero credit. SequenceMatcher threshold ≥ 0.92 required. Mini models cannot inflate scores via keyword pattern matching.
